# منصة سند القانوني برو (Sanad Legal Pro)

هذا المشروع هو منصة متكاملة لإدارة مكاتب المحاماة في المملكة العربية السعودية، معززة بالذكاء الاصطناعي وخبير في الأنظمة العدلية السعودية.

## 🚀 كيفية نشر الموقع (Deployment)

بما أنك قمت ببناء التطبيق باستخدام Next.js، يمكنك نشره مجاناً وبسهولة عبر الطرق التالية:

### الخيار الأول: النشر عبر Netlify (الأسرع)
1. قم بربط مشروعك بحسابك على **GitHub**.
2. اذهب إلى [Netlify Dashboard](https://app.netlify.com/).
3. اختر **"Add new site"** ثم **"Import from Git"**.
4. اختر المستودع (Repository) الخاص بك.
5. سيقوم Netlify تلقائياً باكتشاف أنه مشروع Next.js.
6. اضغط على **"Deploy site"**.

### الخيار الثاني: النشر عبر Firebase App Hosting (الأفضل للتكامل)
1. اذهب إلى [Firebase Console](https://console.firebase.google.com/).
2. أنشئ مشروعاً جديداً.
3. في القائمة الجانبية، اختر **App Hosting**.
4. قم بربط حساب GitHub الخاص بك واختر هذا المشروع.
5. اتبع خطوات الإعداد البسيطة، وسيتم نشر موقعك على رابط `.web.app` مجاناً.

## 🛠 المتطلبات التقنية
- **الذكاء الاصطناعي:** يحتاج النظام إلى مفتاح `GOOGLE_GENAI_API_KEY` لتشغيل مساعد "سند" الذكي. يمكنك وضعه في إعدادات البيئة (Environment Variables) في Netlify أو Firebase.
- **قاعدة البيانات:** يستخدم التطبيق **Firestore** لتخزين بيانات القضايا والموكلين. تأكد من تفعيل "Cloud Firestore" في مشروع Firebase الخاص بك.

## 🇸🇦 مميزات البيئة السعودية
- متوافق بالكامل مع مصطلحات منصة **ناجز** و**نفاذ**.
- مصمم بناءً على **نظام المعاملات المدنية** و**نظام العمل** السعودي الجديد.
- يدعم اللغة العربية بأسلوب قانوني رصين وتنسيق (RTL).
